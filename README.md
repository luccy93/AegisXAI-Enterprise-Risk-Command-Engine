# 🛡️ AegisXAI Enterprise Risk Command Engine

> **AI-Powered Enterprise Customer Churn Intelligence Platform for Fortune 500 Organizations**

AegisXAI is a next-generation Enterprise Risk Command Engine designed to predict, explain, monitor, and mitigate customer churn at enterprise scale.

Inspired by the visual sophistication of Bloomberg Terminal, Tesla Mission Control, Palantir Foundry, and IBM Watson Studio, AegisXAI delivers an executive-grade command center combining Artificial Intelligence, Explainable AI (XAI), predictive analytics, customer intelligence, and operational risk management.

# ⚡ Quick Start

## Clone Repository

```bash
git clone https://github.com/luccy93/AegisXAI-Enterprise-Risk-Command-Engine.git

cd AegisXAI-Enterprise-Risk-Command-Engine
```

## Create Environment

```bash
python -m venv .venv
```

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Launch Platform

```bash
streamlit run app.py
```

Application will be available at:

```text
http://localhost:8501
```
